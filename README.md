API

Others
* [Eslint](https://eslint.org/)
* [Prettier](https://prettier.io/)

## Commands
    git clone https://github.com/YTRodi/paisa-bank.git &&
    cd paisa-bank &&
    yarn &&
    yarn start

## Available scripts
Runs the app in development mode.
```sh		
yarn start
  ```
Android:
```sh		
yarn android
  ```
iOS:
```sh		
yarn ios
  ```
